## General Guidelines

1. **Be Respectful**: Follow our Code of Conduct to ensure a welcoming environment for everyone.
2. **Open an Issue**: For non-trivial changes, please open an issue first to discuss the design.
3. **Small PRs**: We prefer small, focused Pull Requests over large, sweeping changes.

**Thank you for helping make distributed systems more reliable!**
